# Crédits & licences — figures héraldiques

Tous les meubles de l'Atelier du Blason proviennent de **Wikimedia Commons**.
Ils ont été **normalisés** pour l'app (extraction du tracé, mise en teinte plate via
`currentColor` pour suivre l'émail choisi, contour noir conservé). Les sources originales
restent dans `assets/meubles/` ; la version embarquée pour le rendu est dans `meubles.js`
(régénérée par scan dynamique du dossier).

> **Lampe** : dessinée maison (fonction `chargeSVG` dans `app.js`), pas de source externe. Les SVG « lampe » libres trouvés sur Commons sont soit des **illustrations réalistes** qui ne se teintent pas à plat (gradients/multicouleurs figés), soit le mauvais type (lampe de mineur, torche). Le dessin maison — une lampe à huile plate — est le choix héraldique correct (teinte unie).

## Note de licence du livre
Les **sens spirituels** affichés (lecture lumineuse / déchue, n° de page) sont une synthèse
issue de *La Voie du Blason* de **Pascal Gambirasio d'Asseux** — référence pédagogique,
citée comme telle, sans reproduction du texte intégral.

## Ce qui n'est PAS utilisé
Aucune figure de générateurs propriétaires (ex. **coamaker.com**) n'est employée :
seules des sources **domaine public** ou **Creative Commons** (libres) sont intégrées.
