# L'Atelier du Blason

Application web qui met en pratique le livre ***La Voie du Blason — Lecture spirituelle des
armoiries*** de **Pascal Gambirasio d'Asseux** (Éditions Télètes) : composer ses armes
**selon le sens spirituel** de chaque forme, partition, émail et meuble.

> Site statique (HTML / CSS / JS, sans dépendance ni build), publié en GitHub Pages.

## Ce que fait l'app
- **Construction guidée** d'un écu : forme · champ & partition · émaux · meuble central · timbre (cimier, devise).
- **Aperçu SVG en direct**, redessiné à chaque choix.
- **Blasonnement automatique** (la description héraldique en langage propre).
- **Lecture spirituelle** : pour chaque élément, son sens *lumineux* ✦ et *déchu* ⟂, avec le numéro
  de page du livre. C'est la valeur unique de l'app — le guidage que n'offre aucun générateur générique.
- **Garde-fou « règle de tincture »** (ni métal sur métal, ni émail sur émail).
- **Export** de l'écu en SVG + copie du blasonnement et de la lecture.

## Accès
L'atelier s'ouvre derrière un **seuil** (un mot). Saisir le mot, puis *Entrer*.

## Figures héraldiques
Les meubles sont des **SVG libres** (Wikimedia Commons), normalisés pour être teintés selon
l'émail choisi. Détail des sources, auteurs et licences (domaine public / Creative Commons) :
voir **[`CREDITS.md`](CREDITS.md)**. Aucune figure de générateur propriétaire n'est employée.

## Crédit
Les sens spirituels affichés sont une synthèse de *La Voie du Blason* de Pascal Gambirasio
d'Asseux — référence pédagogique, citée comme telle.
